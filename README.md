# MC Tiers

A Minecraft player tier list application by Confession Team.

## Features

- Player tier rankings
- Game mode filtering
- Admin panel for managing players
- Premium and non-premium player support

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Zustand for state management

## Development

```sh
npm install
npm run dev
```

## Deployment

Deploy to Vercel or any static hosting platform.
